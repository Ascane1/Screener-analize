У меня есть торговый бот на Python для Bybit, состоящий из следующих файлов:

- config.py — конфигурация
- main.py — главный модуль
- data_manager.py — работа с API биржи
- screener.py — отбор монет по волатильности
- kernels.py — функции ядер регрессии (Laplace, Gaussian, etc.)
- analyzer.py — анализ и генерация сигналов (Multi Kernel Regression + Relative Performance)
- executor.py — исполнение сделок
- utils.py — утилиты (уведомления, логирование, расчёты)

ЗАДАЧА: Помоги мне протестировать, отладить и запустить бота.

═══════════════════════════════════════════════════════════════
ШАГ 1: ПРОВЕРКА ОКРУЖЕНИЯ
═══════════════════════════════════════════════════════════════

1.1. Проверь, что установлен Python 3.9+
1.2. Создай виртуальное окружение
1.3. Установи зависимости:
     - pybit
     - pandas
     - numpy
     - requests (для уведомлений)

1.4. Проверь структуру файлов проекта

═══════════════════════════════════════════════════════════════
ШАГ 2: НАСТРОЙКА КОНФИГУРАЦИИ
═══════════════════════════════════════════════════════════════

2.1. Настрой config.py:
     - API ключи Bybit (сначала TESTNET=True для тестов)
     - Параметры индикатора (KERNEL_TYPE, BANDWIDTH)
     - Параметры торговли (SL_PERCENT, TP_PERCENT, LEVERAGE)
     
2.2. Проверь, что API ключи имеют нужные разрешения:
     - Чтение позиций
     - Торговля деривативами
     - Чтение баланса

═══════════════════════════════════════════════════════════════
ШАГ 3: ТЕСТИРОВАНИЕ МОДУЛЕЙ (по отдельности)
═══════════════════════════════════════════════════════════════

3.1. Тест data_manager.py:
     - Подключение к API
     - Получение тикеров
     - Получение свечей
     - Получение баланса
     - Получение позиций

3.2. Тест kernels.py:
     - Расчёт kernel regression на тестовых данных
     - Сравнение результата с ожидаемым

3.3. Тест screener.py:
     - Получение топ gainers/losers
     - Проверка фильтрации

3.4. Тест analyzer.py:
     - Расчёт индикаторов
     - Определение Green/Red Zone
     - Генерация сигналов

3.5. Тест executor.py (БЕЗ реального исполнения):
     - Расчёт размера позиции
     - Валидация параметров ордера
